## Electron-react-template

基于 React 和 TypeScript 创建的简单 Electron 模板

## 环境要求

建议 NodeJS 12+ 环境

## 安装

```shell
$ git clone https://github.com/TothingWay/electron-react-template.git
$ cd electron-react-template
$ npm install
```

### 运行

```shell
$ npm run dev  // 端口：3000
```

## 构建运行程序

```shell
$ npm run pack
```

## 构建程序安装包

```shell
$ npm run dist
```
